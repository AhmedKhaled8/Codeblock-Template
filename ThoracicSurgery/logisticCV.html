C. Cross Validation

Cross Validation is a very important technique that is used widely by data scientists. The problem with machine learning models is that you wonâ€™t get to know how well a model performs until you test its performance on an independent data set (the data set which was not used for training the machine learning model). Cross Validation comes to the rescue here and helps you estimate the performance of your model. One type of cross validation is the K-Fold Cross Validation.

i. Implemeting The Cross Validation Class

library(caret)

glm_cv = train(
  form = Death_1yr ~ .,
  data = dataSet,
  trControl = trainControl(method = "cv", number = 10),
  method = "glm",
  family = "binomial"
)

ii. Generating Confidence Interval

Here, we are constructing a 95% confidence interval for our accuracy.

avg_accuracy_glm <- glm_cv$results$Accuracy
accuracy_SD <- glm_cv$results$AccuracySD
min_lr <- avg_accuracy_glm - qnorm(0.975)*accuracy_SD/sqrt(10)
max_lr <- avg_accuracy_glm + qnorm(0.975)*accuracy_SD/sqrt(10)
CI_95_LR = c(min, max)
